# Ecommerce Sales Analytics Dashboard

## Project Overview
This project analyzes ecommerce sales performance using SQL and Power BI.

## Tools Used
- SQL
- Power BI
- Excel
- GitHub

## Key KPIs
- Total Sales
- Total Profit
- Total Orders
- Average Sales

## Analysis Performed
- Sales by Category
- Profit by Category
- Profit Margin by Category
- Region-wise Sales
- Region-wise Profit

## Dashboard Features
- KPI Cards
- Interactive Filters
- Drill Down Analysis
- Business Insights
- Regional Performance Tracking

## Project Files
- Power BI  (.pbix)
- SQL
- Images
- Project Report PDF

## Business Insights
- Identified top-performing categories.
- Compared regional sales and profit trends.
- Evaluated category-wise profit margins.
- Tracked key business KPIs for decision-making.
